## How to use it ?

1. Open signature.html file on your browser
2. * Windows / Linux: Use Ctr + A, Ctr + C
   * MacOS: Use ⌘ + A, ⌘ + C
3. Go to your gmail account and open parameters
4. In the Signature section, create a new one
5. * Windows / Linux: Use Ctr + v
   * MacOS: Use ⌘ + V
6. Save changes

Et voilà !
